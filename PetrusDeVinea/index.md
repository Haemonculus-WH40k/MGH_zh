# Petrus de Vinea Index

彼得罗·德拉·维尼亚（Pietro della Vigna，也作Pier delle Vigne、Petrus de Vineas或de Vineis；约1190年出生于卡普阿， 1249年逝世于圣米尼亚托）是一位意大利法学家和外交家，曾担任皇帝腓特烈二世的宰相和秘书。他被控犯有冒犯君主罪，遭到监禁并被挖去双眼，不久后自杀身亡。他的信件大多以皇帝的名义写成，由 Iselin 出版，其中包含大量关于13世纪历史和文化的宝贵信息。
